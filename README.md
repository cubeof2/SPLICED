# SPLICED

Roll20 character sheet code for the SPLICED roleplaying game system.

## File Descriptions

* **[spliced.html](file:///c:/Users/cubeo/GitHubProjects/SPLICED/spliced.html)**: Contains the HTML layout structure for all tabs (PC, NPC, Vehicle, Inventory, Passives, Notes, Settings) and the Roll20 Sheet Workers (JavaScript block at the bottom of the file) that handle state changes, tab toggles, and roll templates logic.
* **[spliced.css](file:///c:/Users/cubeo/GitHubProjects/SPLICED/spliced.css)**: Contains all the layout styling, theme definitions, grid configurations (CSS Grid layouts for the tabs), and individual component styles.
* **[spliced.json](file:///c:/Users/cubeo/GitHubProjects/SPLICED/spliced.json)**: The metadata manifest file for Roll20, defining file paths, authors, Roll20 user ID, preview image, and custom sheet instructions.
